User

Получение данных о пользователе. Возвращает сущности User и
ContactDetails

  • URL

    /api/user

  • Method:

    GET

  • URL Params

    Required:

    • access_token=[string]

  • Success Response:

    • Code: 200

      Content:
      {
  "queues": [],
  "user": {
    "contact_details": {
      "email": "apploidyakutsk@gmail.com",
      "vkontakteId": 0
    },
    "first_name": "Arthur",
    "id": 62,
    "last_name": "Kupriyanov",
    "username": "Arthur2"
  }
}

  • Error Response:

    • Code: 400 BAD_REQUEST

      Content:
      {
  "error": "invalid_token",
  "error_description": "Your token is invalid. Take authorization and refresh tokens"
}

  • Sample Call:

    GET .../api/user?access_token=QXJ0aHVyMkt1cHJpeWFub3YtMTQ0NTU0MDM3NXNhbHQyNzY=

  • Note

    • Update request to request with response having queue list
