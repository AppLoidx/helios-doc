Helios API Documentation

  • …/auth - аутентификация
  • …/chat - управление чатом очередей
  • …/note - управление заметками пользователей
  • …/queue - создание\удаление\присоединение к очереди
  • …/queue/{queue_name} - управление очередью
  • …/register - регистрация
  • …/user - данные о пользователе
  • …/check - проверка данных, таких как существование пользователей или приватность очередей

Основные положения

Для обращения к API Helios нужно использовать хост https://helios-service.herokuapp.com по пути /api:

https://helios-service.herokuapp.com/api/METHOD_NAME

OAuth

Сейчас используется не полная модель OAuth 2.0. С помощью метода auth необходимо получить access_token и refresh_token. Для этого необходимо отправить запрос вместе с логином и паролем пользователя.
